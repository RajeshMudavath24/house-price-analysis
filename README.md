# House Price Analysis and Prediction

## Overview
This project focuses on analyzing housing data and building a simple model to predict house prices.

## Dataset
California Housing dataset.

## What I did
- Loaded the dataset using pandas
- Checked structure, columns, and missing values
- Cleaned the data by removing missing entries
- Performed basic statistical analysis
- Analyzed correlation between features
- Visualized income vs house price
- Built a simple Linear Regression model

## Key Observations
- Median income has a strong impact on house prices
- Location (latitude and longitude) also affects pricing
- Some missing values were present and handled

## Model
- Used Linear Regression
- Split data into training and testing sets
- Evaluated model using score

## Tools Used
- Python
- Pandas
- Matplotlib
- Scikit-learn
